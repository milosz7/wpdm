Welcome to the TSProto documentation
---------------------------------



**WPDM (Warsztat Praktyka Data Mining)** is toolkit for usefull functions and models used during WPDM course at JU..

Install
=======

LUX can be installed from either `PyPI <https://pypi.org/project/wpdm>`_ or directly from source code `GitHub <https://github.com/sbobek/wpdm>`_

To install form PyPI::

   pip install milosz7wpdm

To install from source code::

   git clone https://github.com/milosz7/wpdm
   cd wpdm
   pip install .
